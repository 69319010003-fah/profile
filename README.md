## 🖥️ รายวิชา Backend Development & GitHub

ในการเรียนรู้รายวิชานี้ **GitHub** เปรียบเสมือนเครื่องมือชิ้นสำคัญที่ขาดไม่ได้ โดยมีบทบาทหลักๆ ดังนี้:

### 1. GitHub คืออะไร?
GitHub คือแพลตฟอร์มบนเว็บที่ใช้ระบบ **Git** (Version Control) ในการจัดเก็บและจัดการซอร์สโค้ดของโปรเจกต์ ช่วยให้นักพัฒนาสามารถติดตามการเปลี่ยนแปลงของโค้ด ย้อนกลับไปดูเวอร์ชันเก่าๆ และทำงานร่วมกับผู้อื่นได้อย่างเป็นระบบ

### 2. ความสำคัญในสายงาน Backend
* **Collaboration:** ช่วยให้ทำงานร่วมกับทีมพัฒนาคนอื่นๆ ได้อย่างไร้รอยต่อผ่านระบบ Pull Request และ Code Review
* **Backup & Deployment:** โค้ดไม่หาย และสามารถเชื่อมต่อไปยังระบบ Cloud (เช่น Heroku, AWS, Vercel) เพื่อนำระบบขึ้นใช้งานจริงได้ง่าย
* **Portfolio:** เป็นพื้นที่แสดงผลงานและฝีมือการเขียนโค้ดให้กับอาจารย์และผู้ประกอบการในอนาคต

## 📦 ตัวอย่างการเขียน Code ด้วย Node.js

นี่คือตัวอย่างการสร้าง Web Server เบื้องต้นด้วย **Node.js** เพื่อเริ่มต้นทำความเข้าใจการทำงานของ Backend

### 📄 โค้ดเริ่มต้น (Basic HTTP Server)

```javascript
// นำเข้าโมดูล http ซึ่งเป็นโมดูลพื้นฐานของ Node.js
const http = require('http');

const hostname = '127.0.0.1';
const port = 3000;

// สร้าง Server และกำหนดการตอบกลับ (Response)
const server = http.createServer((req, res) => {
  res.statusCode = 200; // ส่ง Status Code 200 (OK) กลับไป
  res.setHeader('Content-Type', 'text/plain; charset=utf-8');
  res.end('สวัสดีชาวไอที พณิชยการบางนา! 👋\nยินดีต้อนรับสู่ระบบ Backend ของฉัน');
});

// สั่งให้ Server เริ่มทำงานที่ Port ที่กำหนด
server.listen(port, hostname, () => {
  console.log(`Server กำลังทำงานที่ http://${hostname}:${port}/`);
});
